# NoteCycle

🔗 **Live Website:** [notecycle.vercel.app](https://notecycle.vercel.app)

NoteCycle is a full-stack data-driven note-sharing platform built to connect students with high-quality study resources. Developed with Next.js, TypeScript, and Supabase, the project demonstrates core data science competencies — structured data modeling, user behavior tracking, and an engagement ranking system designed to surface insights on resource quality, student demand, and course popularity.

## Data & Engineering Highlights

- Designed a relational data schema in Supabase (PostgreSQL) to organize notes by course and professor, enabling structured querying and aggregation across user-generated content
- Built an upvote/downvote ranking system that captures interaction-level data, laying the groundwork for engagement scoring, quality metrics, and demand forecasting by course
- Implemented user activity tracking across upload, browse, and vote events — structured to support future analysis of usage patterns and resource effectiveness
- Architected a full-stack pipeline from data ingestion (note uploads) to storage, retrieval, and ranked display — mirroring real-world data workflows

## Features

- User authentication
- Upload and browse notes organized by course and professor
- Upvote/downvote ranking system
- Dashboard and profile pages

## Tech Stack

Next.js · React · TypeScript · Supabase (PostgreSQL) · Tailwind CSS
