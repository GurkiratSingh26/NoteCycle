# NoteCycle

🔗 **Live Website:** [notecycle.vercel.app](https://notecycle.vercel.app)

NoteCycle is a full-stack note-sharing platform where students can upload, browse, and rank lecture notes by course and professor. Built with Next.js, TypeScript, and Supabase, the project demonstrates data-focused skills through structured data organization, user activity tracking, and a ranking system that can support analysis of engagement, resource quality, and student demand.

## Features

- User authentication
- Upload and browse notes
- Organize notes by course and professor
- Upvote and downvote system
- Dashboard and profile pages

## Tech Stack

- Next.js
- React
- TypeScript
- Supabase
- Tailwind CSS

## Use Case

NoteCycle gives students one place to share and discover useful study resources. It also shows how user-generated data and interaction data can be organized to support future analysis of usage patterns, popular courses, and note quality.
